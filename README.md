🔴 CPU 异常监控器

防止在不知情的情况下沦为“肉鸡”

当你的电脑被植入后门、挖矿木马等恶意程序时，CPU 占用往往会异常飙升。
本工具像一个 🛡️ 哨兵，实时盯着 CPU 占用率，一旦超过你设定的警戒线，立刻弹出 强制置顶 警告，并自动揪出高占用进程的详细文件信息，
让你一眼就能看出谁在搞鬼。

---

❓ 为什么需要它？

 🖥️ 电脑突然变卡，风扇狂转，你以为只是正常高负载，
    但很可能 已被远程控制 或 正在被偷偷挖矿。
 🔍 很多恶意软件会故意隐藏窗口，任务管理器也难以察觉，
    而 CPU 占用率异常是最直接的“身体信号”。
 ⚡ 本工具就是帮你 第一时间发现异常，定位可疑进程，
    避免长期充当肉鸡而浑然不知。

-------------------------------------------

✨ 主要功能

功能 说明
🔔 强制置顶弹窗 无论你在打游戏、看视频还是全屏应用，警告框永远在最前面
🔍 自动进程侦查 列出占用前五的进程：程序名 / CPU % / 文件大小 / 创建时间 / 完整路径
⚠️ 可疑路径标记 位于临时文件夹（Temp, Downloads）的进程自动标为可疑
📋 实时异常日志 主界面内以整齐表格记录每次警告，一目了然
💾 按需导出日志 日常不生成任何文件，点击 导出日志 时才保存
🚀 开机自启 + 托盘 勾选后开机自动以托盘图标运行，无感守护
🎨 暗黑主题 黑底红字，酷炫又护眼

🚀 使用方法

▶️ 第一步：运行程序

直接双击 CPU监控.exe 即可启动。

 如果弹出 UAC（用户账户控制）窗口，请点击 “是”（需要管理员权限来设置开机自启和获取完整进程路径）。
 启动后程序会最小化到系统托盘，不会打扰你当前的工作或游戏。
 右键点击托盘图标可以选择 “显示窗口” 或 “退出”。

⚙️ 第二步：设置阈值并启动监控

1. 在主界面的 “CPU 阈值 (%)” 中输入你想要的警戒线（例如 80）。
2. 点击 🚀 启动监控。
3. 当前 CPU 占用会实时显示在界面右上角。

🔔 第三步：异常告警

 >当 CPU 占用 ≥ 设定阈值 时：
  > 弹出 强制置顶 警告框，显示当前占用率和可疑进程。
   >主界面下方 “异常日志” 会以表格形式列出进程的详细文件信息。
 日常无需关注，需要查看时打开主窗口即可。

💾 第四步：导出日志

 >点击主界面上的 💾 导出日志，选择保存路径，即可将当前日志保存为文件（日常不占用硬盘空间）。

⏹️ 退出程序

 右键托盘图标 → “退出”，即可彻底关闭监控。
 直接关闭主窗口只会最小化到托盘，监控继续运行。
